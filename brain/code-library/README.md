# Code Library
Reusable VWO-ready JS/CSS snippets. Claude adapts selectors and styles to each client context file.
## Available Snippets
| File | Pattern |
|---|---|
| sticky-mobile-cta.js | Sticky CTA on mobile scroll |
| social-proof-inject.js | Inject trust signal near CTA |
| package-badge.js | Add Most Popular badge to pricing tier |
| urgency-banner.js | Top-of-page urgency strip |
| form-step-progress.js | Progress indicator on multi-step forms |
| hero-cta-simplify.js | Replace multiple CTAs with single primary |
## Adding A Snippet
1. Create .js file in this folder
2. Add row to table above
3. Include at top of every file:
// Snippet: [name]
// Purpose: [what it does]
// Selectors to update: [list per client]
// Dependencies: [jQuery / vanilla JS / VWO API]
// Tested on: [clients]
Every winning variation gets abstracted into a snippet for reuse.
